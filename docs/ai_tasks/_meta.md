## Workflow

1. **Read** → Open the task file (basic or advanced)
2. **Execute** → Follow action steps in `.task_actions/[level]/[task-id]-action-[level].md`
3. **Verify** → Test the implementation manually
4. **Document** → Update the tutorial file if specified
5. **Next** → Move to the next task (user mannual approval is needed)

## Rules

- Install popular NPM packages if needed instead of creating utilities on your own
- One task at a time
- Test before proceeding
- Update tutorials when indicated
