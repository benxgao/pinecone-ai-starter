# Embeddings

## What

Embedding is a numerical vector representation of text that captures semantic meaning.

---

## Why

- Enables semantic search (not keyword match)
- Allows similarity comparison between texts
- Foundation for vector databases and RAG systems

## Trade-offs

Pros:
- Captures meaning beyond keywords
- Works well for fuzzy matching

Cons:
- Requires external API (cost + latency)
- Not human-readable

Alternatives:
- Keyword search (BM25)
- Hybrid search (vector + keyword)
